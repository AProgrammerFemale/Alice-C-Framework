# Alice C++ Framework

Modern General Purpose C++ Framework (Clang & GCC Only)

Linux:

```C++
git clone https://github.com/AProgrammerFemale/Alice-C-Framework.git
cd Alice-C-Framework
mkdir Build
cd Build
cmake ..
make
```

Windows:

```C++
git clone https://github.com/AProgrammerFemale/Alice-C-Framework.git
cd Alice-C-Framework
mkdir Build
cd Build
cmake .. -G "MinGW Makefiles"
make
```
